# Real-Time Chat Application

A full-stack real-time messaging application built using React.js, Node.js, Express.js, Socket.io, and MongoDB. The application enables users to communicate instantly with persistent chat history and responsive user experience.

---

## 🚀 Features

- Real-time messaging using Socket.io
- Persistent chat history with MongoDB
- Responsive modern UI
- Secure backend API
- Scalable client-server communication
- Live message delivery
- RESTful API integration
- Full-stack MERN architecture

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- Socket.io

### Database
- MongoDB
- Mongoose ODM

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/mekuannent-biyazn/real-time-chat-app.git
cd real-time-chat-app
```

---

### 2. Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

---

## 🔐 Environment Variables

Create a `.env` file inside the `server` folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

---

## ▶️ Run the Application

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm start
```
